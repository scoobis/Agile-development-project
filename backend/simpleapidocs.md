# Project Backend REST API Endpoints
[RESTful API guidelines](https://restfulapi.net/)

- GET = Read existing
- POST = Create new 
- PUT = Update existing
- DELETE = delete existing

## User
| Done | Method | Endpoint |
| ---- | ------ | -------- |
| Y | POST | /user/register |
| Y | POST | /user/login |
| - | GET | /user/{id} |
| - | PUT | /user/{id} |
| - | DELETE | /user/{id} |
